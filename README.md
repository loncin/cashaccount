# 共享记账 (Shared Bookkeeping)

一款基于微信小程序云开发的共享记账应用，旨在提供简洁、高效、可协作的财务管理体验。

## 🌟 主要功能

- **多人共享**：支持创建多个共享账本，通过分享邀请好友加入，实现多人共同记账。
- **快速记账**：直观的记账界面，支持支出与收入切换，可记录金额、分类、成员、备注，并上传凭证图片。
- **智能 OCR 识别**：集成云开发 OCR 能力，支持拍照或截图识别账单金额、日期等关键信息。
- **多维度统计**：提供分类统计、每日趋势等图表，直观了解收支构成。
- **账户管理**：支持管理多个账户（如现金、支付宝、银行卡等），自动根据账单计算实时余额。
- **周期记账**：支持设置每天、每周、每月自动生成的周期性账单，省去重复输入。
- **预算系统**：设定月度预算，记账时若触发超支将自动弹出提醒。
- **数据导出**：支持将账单明细导出为 CSV 格式，方便进一步分析。

## 🛠️ 技术栈

- **前端**：微信小程序原生开发 (WXML, WXSS, JavaScript)
- **后端**：微信云开发 (CloudBase)
  - **云函数**：核心业务逻辑处理 (`cloudApi`)
  - **云数据库**：NoSQL 数据库存储数据
  - **云存储**：账单凭证图片存储
  - **OCR 服务**：智能票据识别

## 🚀 快速开始

1. **环境准备**：
   - 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
   - 开通微信小程序账号并获取 AppID。

2. **导入项目**：
   - 打开微信开发者工具，点击“导入”，选择当前文件夹。

3. **配置云开发**：
   - 点击工具栏左上角的“云开发”按钮，开通云环境。
   - 获取你的 **环境 ID**。
   - 打开 `app.js`，将 `wx.cloud.init` 中的 `env` 替换为你自己的环境 ID。

4. **部署云函数**：
   - 在 `cloudfunctions` 目录下，右键点击 `cloudApi` 文件夹，选择“上传并部署：所有文件”。
   - 如果使用了 OCR 功能，请同样部署 `ocrAction` 云函数。
